Guia basica de git

Git es una herramienta de control de versiones

Una herramienta de control de versiones nos permite tener multiples puntos de guardado tanto adelante como detras de el estado actual de nuestro proyecto, ademas tambien podemos tener versiones alternas simultaneas de el mismo proyecto o archivo.

Esto puede sonar un poco complicado pero al ir a la practica se hara mas facil de entender.

Prerequisitos

Antes de iniciar hay ciertas herramientas qeu debemos preparar para hacer los ejercicios con facilirdad.

  • Crear una cuenta de GitHub aqui.
